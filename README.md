# Tic-Tac-Toe Pro

A modern and interactive Tic-Tac-Toe game developed using **HTML5, CSS3, and JavaScript**. The application offers both multiplayer and AI gameplay with multiple difficulty levels, a responsive interface, live score tracking, and theme switching.

---

## Features

- Player vs Player mode
- Player vs Computer mode
- Three AI difficulty levels
  - Easy
  - Medium
  - Hard (Minimax Algorithm)
- Choose the starting player
- Live scoreboard
- Win detection with highlighted winning combination
- Draw detection
- Play Again option
- Main Menu navigation
- Dark and Light themes
- Clean and responsive user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## AI Levels

### Easy
Computer selects a random available move.

### Medium
Computer attempts to win if possible or blocks the opponent before making a random move.

### Hard
Computer uses the Minimax Algorithm to determine the optimal move, making it extremely difficult to defeat.
